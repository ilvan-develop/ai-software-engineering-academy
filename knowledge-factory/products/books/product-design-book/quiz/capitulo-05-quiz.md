# Quiz — Capítulo 5: UI Design

**Instruções:** 5 perguntas de múltipla escolha. Apenas uma alternativa correta.

---

## Pergunta 1 (Fácil)

O que significa o acrônimo CRAP em design visual?

- A) **Contraste, Repetição, Alinhamento, Proximidade**
- B) Cor, Ritmo, Acessibilidade, Proporção
- C) Criatividade, Relevância, Agilidade, Precisão
- D) Cor, Repetição, Alinhamento, Padrão

**Resposta:** A

---

## Pergunta 2 (Fácil)

Qual o ratio mínimo de contraste para texto normal segundo WCAG AA?

- A) 3:1
- B) **4.5:1**
- C) 7:1
- D) 2:1

**Resposta:** B

---

## Pergunta 3 (Médio)

Qual a vantagem de usar uma escala modular (ex: 1.25) para tipografia?

- A) **Garantir proporções harmônicas e consistentes entre os tamanhos**
- B) Reduzir o número de estilos de texto disponíveis
- C) Facilitar a implementação em CSS
- D) Aumentar o contraste entre títulos e corpo

**Resposta:** A

---

## Pergunta 4 (Médio)

O que significa "fidelidade de implementação" no contexto de UI para devs?

- A) **O grau em que o código implementado corresponde fielmente ao design especificado**
- B) A qualidade do código TypeScript escrito
- C) O número de componentes reutilizados
- D) A velocidade de carregamento da página

**Resposta:** A

---

## Pergunta 5 (Difícil)

Por que dark mode NÃO é simplesmente inverter cores?

- A) Porque é mais fácil implementar com fundo preto puro
- B) **Porque a percepção de contraste muda no escuro, exigindo ajustes específicos**
- C) Porque usuários preferem cores vibrantes no dark mode
- D) Porque a acessibilidade não se aplica ao dark mode

**Resposta:** B
