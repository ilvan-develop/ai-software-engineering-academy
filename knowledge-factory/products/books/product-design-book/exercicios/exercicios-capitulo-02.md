# Exercícios — Capítulo 2: Design Thinking

> **Progressão:** Fácil → Médio → Difícil  
> **Total:** 4 exercícios

---

## Exercício 1 — Fácil: As 5 Etapas

**Tema:** Etapas do Design Thinking

Ordene as 5 etapas do Design Thinking na sequência correta:

- ( ) Prototipação
- ( ) Teste
- ( ) Ideação
- ( ) Definição
- ( ) Empatia

**Resposta:** Empatia → Definição → Ideação → Prototipação → Teste

---

## Exercício 2 — Médio: How Might We

**Tema:** Transformar problemas em oportunidades

Reescreva cada problema como uma pergunta "How Might We" (Como Poderíamos):

| Problema | How Might We |
|----------|-------------|
| "Usuários não conseguem encontrar o botão de salvar" | ? |
| "O onboarding tem 60% de abandono" | ? |
| "Clientes demoram 3 dias para receber suporte" | ? |
| "Usuários não confiam na plataforma para pagamentos" | ? |

**Exemplo:** "Usuários esquecem a senha" → "HMW tornar o login seguro sem depender de senha?"

---

## Exercício 3 — Médio: Crazy 8

**Tema:** Geração rápida de ideias

Problema: **Pessoas idosas têm dificuldade em usar apps de banco** (fontes pequenas, fluxos complexos, medo de golpes).

**Tarefa:** Gere 8 ideias diferentes (1-2 linhas cada) para resolver este problema. Pense em soluções analógicas, digitais e de serviço. Não se preocupe com viabilidade — quantity over quality.

---

## Exercício 4 — Difícil: Workshop de Design Thinking

**Tema:** Facilitar um workshop

Projete uma workshop de **4 horas** para um time de produto (PM + 2 devs + 1 designer) resolver o desafio:

> "Como melhorar a experiência de devolução de produtos em um e-commerce?"

**Estrutura:**
- **0:00–0:30** — Aquecimento e alinhamento
- **0:30–1:30** — Empatia (atividade prática com personas)
- **1:30–2:30** — Ideação (técnica de geração de ideias)
- **2:30–3:15** — Prototipação (baixa fidelidade)
- **3:15–4:00** — Teste e validação

Para cada bloco, descreva:
- Objetivo da atividade
- Materiais necessários
- Entregável esperado
- Como evitar vieses comuns (designer só aceita ideias de UI, dev fala "isso é difícil")
