# Como usar este livro

## Estrutura

Este livro está organizado em **6 capítulos** que seguem a progressão natural do design de produto digital:

```text
Capítulo 1: Product Discovery     → O problema certo
Capítulo 2: Design Thinking       → Ideação estruturada
Capítulo 3: UX Research & Design  → Usuário no centro
Capítulo 4: Wireframes e Proto.   → Do papel ao clique
Capítulo 5: UI Design             → A camada visual
Capítulo 6: Design System         → Consistência em escala
```

## Pré-requisitos

- Conhecimento básico de HTML, CSS e JavaScript/TypeScript
- Familiaridade com React (para os capítulos 5 e 6)
- Node.js 18+ instalado (para exemplos práticos)

## Material complementar

Todo o código fonte dos exemplos está disponível no repositório da AI Software Engineering Academy. Os designs e protótipos mencionados podem ser acessados nos links fornecidos em cada capítulo.
