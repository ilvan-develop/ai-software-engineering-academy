# Quiz — Capítulo 4: Segurança

**Instruções:** 5 perguntas de múltipla escolha. Apenas uma alternativa correta.

---

## Pergunta 1 (Fácil)

Qual é o algoritmo de hash RECOMENDADO para armazenar senhas de forma segura?

- [ ] A) MD5
- [ ] B) SHA-256
- [ ] C) **bcrypt**
- [ ] D) Base64

**Resposta:** C

---

## Pergunta 2 (Fácil)

O que significa a sigla JWT?

- [ ] A) JavaScript Web Token
- [ ] B) **JSON Web Token**
- [ ] C) Java Web Transport
- [ ] D) JSON with Time

**Resposta:** B

---

## Pergunta 3 (Médio)

No padrão RBAC, o que é verificado para autorizar uma ação?

- [ ] A) O IP de origem da requisição
- [ ] B) **O role (papel) associado ao usuário autenticado**
- [ ] C) O tempo decorrido desde o login
- [ ] D) O dispositivo usado na requisição

**Resposta:** B

---

## Pergunta 4 (Médio)

Qual das seguintes é uma vulnerabilidade OWASP Top 10 relacionada a queries de banco de dados?

- [ ] A) Cross-Site Scripting (XSS)
- [ ] B) **Injection (SQL, NoSQL, OS)**
- [ ] C) Broken Authentication
- [ ] D) Security Misconfiguration

**Resposta:** B

---

## Pergunta 5 (Difícil)

Por que refresh tokens rotativos são mais seguros que refresh tokens fixos?

- [ ] A) **Porque um refresh token comprometido só é válido uma vez, limitando o dano**
- [ ] B) Porque são mais rápidos de gerar
- [ ] C) Porque não precisam de armazenamento no servidor
- [ ] D) Porque usam criptografia simétrica

**Resposta:** A
