# Certificação — Capstone (Projeto Final)

## Prova Objetiva (20 questões)

**Tempo estimado:** 60 minutos  
**Mínimo para aprovação:** 70% (14/20)  
**Pré-requisito:** As 5 certificações anteriores

## Prova Prática — Projeto Final Integrado

**Título:** Sistema Completo Enterprise

**Descrição:** Desenvolva um sistema completo com:
1. Product Discovery — pesquisa, personas, jornada
2. Backend — DDD, CQRS, Event Sourcing, multi-tenant
3. Frontend — Next.js com SSR + design system
4. DevOps — Docker, CI/CD, Kubernetes, observabilidade
5. Governança — ADRs, BQS, auditoria de qualidade

**Critérios:** Escopo e definição (10%), Arquitetura e modelagem (25%), Implementação (30%), Testes (15%), DevOps (10%), Documentação (10%)
