# Template de Entrega

> Use este template para estruturar a entrega do projeto.

---
## Checklist de Auto-Avaliacao

- [ ] Roteiro de entrevista bem estruturado (sem perguntas indutoras)
- [ ] Hipóteses no formato correto com métricas testáveis
- [ ] Opportunity Solution Tree completa (outcome → oportunidades → soluções → experimentos)
- [ ] User Story Mapping com corte de MVP claro
- [ ] Lean Canvas e Value Proposition Canvas preenchidos
- [ ] Proto-personas baseadas em evidência (não suposição)
- [ ] User stories com critérios de aceite
- [ ] Opportunity Backlog priorizado com evidência
- [ ] Resumo executivo convincente para stakeholders
- [ ] Experimentos com métricas de sucesso e critérios go/no-go
