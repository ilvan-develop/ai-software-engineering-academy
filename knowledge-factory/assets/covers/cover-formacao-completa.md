# Cover Prompt — Formação Completa Enterprise

## Conceito

Livro abrangente sobre engenharia de software enterprise. Capa que comunica: totalidade, estrutura, solidez técnica.

## Briefing Visual

**Estilo:** Clean, moderno, tech-enterprise. Vermelho coral (#E04A4A) como cor dominante, gradiente sutil para tons mais escuros.

**Elementos:**
- Foco: Diagrama de arquitetura abstrato 3D no centro — camadas sobrepostas (frontend, backend, dados, infra) em wireframe translúcido
- Background: Fundo escuro (#1A1A2E) com grid fino de linhas de circuito/pipeline
- Tipografia: Título em branco, sans-serif bold (Inter ou SF Pro Display)
- Subtítulo: "Engenharia de Software Enterprise — 27 Módulos" em cinza claro
- Badge: "6 Trilhas Completas" canto superior direito

