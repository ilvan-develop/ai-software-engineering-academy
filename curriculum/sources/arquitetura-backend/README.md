# Arquitetura Backend

**A espinha dorsal do sistema.** Arquitetura, modelagem, APIs, segurança e multi-tenancy.

## Objetivos

- Projetar arquiteturas modulares com Clean Architecture, DDD e SOLID
- Modelar dados com Prisma e PostgreSQL (ERD, UML, diagramas)
- Construir APIs REST escaláveis com NestJS
- Implementar segurança: OWASP, JWT, OAuth, rate limiting
- Projetar sistemas multi-tenant com diferentes estratégias de isolamento

## Público-alvo

Desenvolvedores backend que querem evoluir de "fazer API" para "arquitetar sistemas".

## Pré-requisitos

- Curso [Fundamentos Enterprise](../fundamentos-enterprise/README.md)
- Experiência com Node.js e TypeScript
