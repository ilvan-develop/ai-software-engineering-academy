# Agentes para o Módulo 10

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar conteúdo de backend |
| Technical Writer | Escrever exemplos reais de código NestJS |
| Reviewer | Validar boas práticas e padrões |
| Backend Expert Agent | Referência prática |

## Instruções específicas

### Curriculum Architect

Ao planejar:
- Conectar com Módulo 08 (Arquitetura) — aplicar Clean Arch no NestJS
- Conectar com Módulo 12 (Segurança) — guards, auth, rate limiting
- Mostrar a progressão: estrutura → implementação → testes

### Technical Writer

Ao escrever:
- Código real de NestJS que funciona (não pseudo-código)
- Incluir imports e decorators completos
- Mostrar o padrão DDD aplicado no NestJS
- Incluir exemplos de testes
