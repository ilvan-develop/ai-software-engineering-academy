# Agentes para o Módulo 12

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar conteúdo de segurança |
| Technical Writer | Escrever com exemplos práticos |
| Reviewer | Validar precisão das recomendações |
| Security Expert Agent | Referência prática para este módulo |

## Instruções específicas

### Curriculum Architect

Ao planejar este módulo:
- Conectar com o Módulo 19 (Auditorias) — a auditoria de segurança é a mais crítica
- Mostrar exemplos reais de ataques e suas consequências
- Enfatizar que segurança é prática diária, não evento único

### Technical Writer

Ao escrever:
- Código "ruim" seguido de código "bom" para cada conceito
- Usar a stack da formação (NestJS + Prisma + TypeScript)
- Incluir exemplos de configuração real (Helmet, Throttler, CASL)
- Mostrar os headers HTTP reais que o Helmet configura

### Reviewer

Ao revisar:
- Todas as recomendações de segurança estão atualizadas?
- Os exemplos "ruins" realmente representam vulnerabilidades?
- As configurações propostas são seguras para produção?
- Não há contradições com o módulo de Auditoria?
