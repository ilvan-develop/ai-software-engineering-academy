# Fundamentos Enterprise

**A base de tudo.** Como pensar, estruturar e decidir como um arquiteto enterprise.

## Objetivos

- Entender o papel do arquiteto, PO e IA no desenvolvimento moderno
- Desenvolver mentalidade enterprise: escalabilidade, governança, segurança, compliance
- Aprender a dividir trabalho entre humanos e agentes de IA

## Público-alvo

Desenvolvedores que estão migrando de projetos individuais/small-team para sistemas enterprise com múltiplos stakeholders, requisitos não-funcionais e equipes distribuídas.

## Pré-requisitos

- Experiência básica com desenvolvimento de software (1-2 anos)
- Conhecimento básico de arquitetura de sistemas
