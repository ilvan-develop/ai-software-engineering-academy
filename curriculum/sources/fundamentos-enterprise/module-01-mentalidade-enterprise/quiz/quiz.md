# Quiz — Módulo 01

## Pergunta 1

Qual a principal diferença entre software de consumo e software Enterprise?

- a) Enterprise usa tecnologias diferentes
- b) Enterprise é construído para organizações, com requisitos de multi-usuário, segurança e governança
- c) Enterprise é sempre mais caro
- d) Enterprise não precisa de testes

**Resposta:** b

---

## Pergunta 2

Qual estratégia de escalabilidade é mais sustentável a longo prazo?

- a) Vertical (aumentar recursos da máquina)
- b) Horizontal (adicionar mais máquinas)
- c) Nenhuma, sistemas não escalam
- d) Depende apenas do orçamento

**Resposta:** b

---

## Pergunta 3

O que é governança em desenvolvimento de software?

- a) Um cargo na empresa
- b) Conjunto de regras e processos para garantir consistência e qualidade
- c) Uma ferramenta de monitoramento
- d) Um tipo de banco de dados

**Resposta:** b

---

## Pergunta 4

Qual dos 3 pilares da observabilidade rastreia o fluxo de uma requisição através dos serviços?

- a) Logs
- b) Métricas
- c) Tracing
- d) Alertas

**Resposta:** c

---

## Pergunta 5

Qual estratégia de isolamento multi-tenant oferece o maior nível de segurança?

- a) Row-Level Security (RLS)
- b) Schema per Tenant
- c) Database per Tenant
- d) Compartilhamento total

**Resposta:** c

---

## Pergunta 6

Quantos "noves" de disponibilidade um sistema Enterprise crítico normalmente precisa?

- a) 99% (dois noves)
- b) 99.99% (quatro noves)
- c) 90% (um nove)
- d) 50%

**Resposta:** b

## Pergunta 7

Qual destas NÃO é uma exigência da LGPD?
