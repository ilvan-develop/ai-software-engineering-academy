# Agentes para o Módulo 01

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar os conceitos fundamentais |
| Technical Writer | Explicar cada conceito com exemplos práticos |
| Reviewer | Validar precisão técnica |

## Instruções específicas

### Curriculum Architect

Ao planejar este módulo:
- Conectar cada conceito a um problema real que o aluno já enfrentou
- Mostrar a interdependência entre os conceitos
- Preparar o aluno para aplicar a mentalidade Enterprise nos módulos seguintes

### Technical Writer

Ao escrever este módulo:
- Tom direto e prático (não acadêmico)
- Cada conceito deve vir acompanhado de "o que acontece se ignorar isso"
- Usar o mínimo de jargão possível — explicar cada termo na primeira aparição
- Incluir exemplos de código reais (Prisma, TypeScript, NestJS) onde fizer sentido

### Reviewer

Ao revisar este módulo:
- Os conceitos estão na ordem lógica correta?
- As explicações são precisas tecnicamente?
- Os exemplos de código estão corretos e seguem boas práticas?
- O tom é adequado para um desenvolvedor que está migrando para Enterprise?
