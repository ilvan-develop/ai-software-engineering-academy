# Módulo 00 — Introdução

**Antes de escrever uma linha de código.**

## Objetivos

- Entender como a IA mudou o desenvolvimento de software
- Comparar desenvolvimento tradicional vs. desenvolvimento com agentes
- Definir o papel do arquiteto, do Product Owner e da IA
- Aprender a dividir trabalho entre humanos e agentes

## Competências

**Hard skills:**
- Modelos de colaboração humano-IA
- Estruturação de equipes de desenvolvimento com agentes

**Soft skills:**
- Pensamento crítico sobre o papel da IA
- Decisão sobre o que delegar vs. o que reter
