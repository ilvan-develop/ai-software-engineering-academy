# Agentes para o Módulo 00

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar o conteúdo introdutório |
| Technical Writer | Escrever a aula em linguagem didática |
| Reviewer | Validar clareza e precisão do conteúdo |

## Instruções específicas

### Curriculum Architect

Ao planejar este módulo, foque em:
- Contextualizar a transformação causada pela IA
- Estabelecer a mentalidade correta (não é sobre ferramentas, é sobre processos)
- Preparar o aluno para os próximos 20 módulos
- Introduzir a Matriz de Delegação como ferramenta central

### Technical Writer

Ao escrever este módulo:
- Tom inspirador e provocativo
- Exemplos concretos e dados reais (Stack Overflow Survey)
- Evitar jargão técnico profundo (módulo introdutório)
- Usar analogias do mundo real (arquiteto de construção, regente de orquestra)

### Reviewer

Ao revisar este módulo:
- O conteúdo estabelece a base correta para os módulos seguintes?
- As metáforas e analogias são precisas?
- Os dados citados (63%, 40%, ~3%) são referenciados corretamente?
- Há consistência terminológica com o restante da formação?
