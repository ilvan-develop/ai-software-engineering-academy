# Projeto Módulo 00 — Visão de um Sistema Enterprise

## Objetivo

Criar a visão completa de um sistema Enterprise, definindo o escopo, agentes envolvidos e o ciclo de desenvolvimento.

## Contexto

Você é o arquiteto de uma startup que vai construir um **SaaS de Gestão de Projetos Enterprise**. O sistema precisa atender empresas de médio porte (50-500 funcionários) com suporte a múltiplos times, projetos, tarefas e relatórios.

## Entregáveis

### 1. Problem Statement

Defina em 3-5 frases:
- Qual problema o sistema resolve
- Para quem
- Qual o diferencial competitivo

### 2. Stakeholders e Personas

Identifique:
- Quem usa o sistema (personas)
- Quem paga pelo sistema
- Quem mantém o sistema

### 3. Stack Tecnológica

Escolha a stack e justifique cada escolha:
- Frontend
- Backend
- Banco de dados
- Infraestrutura
- IA/Ferramentas

### 4. Matriz de Delegação

Para cada funcionalidade principal, classifique:
- **Humano decide**
- **Humano + Agente**
- **Agente faz**
- **Agente decide**

### 5. Ciclo de Desenvolvimento

Mapeie as 10 etapas do ciclo de desenvolvimento, identificando para cada:
- Agente responsável
- Artefato produzido
- Critério de aceite

### 6. Regras Iniciais para Agentes (AGENTS.md)

Escreva as primeiras 10 regras que os agentes devem seguir neste projeto.
