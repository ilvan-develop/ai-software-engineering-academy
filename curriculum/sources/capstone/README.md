# Capstone — Projeto Final

**Colocar tudo em prática.** Construir um SaaS Enterprise completo, do discovery ao deploy com observabilidade.

## Objetivos

- Integrar todos os 20 módulos anteriores em um único projeto
- Tomar decisões arquiteturais reais com trade-offs documentados (ADRs)
- Implementar com agentes de IA e auditar continuamente
- Entregar um sistema pronto para produção

## Projeto

Construir uma plataforma SaaS de gestão de projetos/tarefas com:
- Autenticação multi-tenant
- CRUD com permissões baseadas em papéis
- Notificações em tempo real
- Relatórios e dashboards
- Integração com pagamentos
- Deploy com observabilidade completa

## Critérios de avaliação

- Arquitetura e decisões documentadas (ADRs)
- Código modular e testado
- Pipeline CI/CD funcional
- Dashboard de observabilidade
- Apresentação final com demo
