# Governança & Automação

**Qualidade, controle e inteligência.** Como governar o desenvolvimento, auditar continuamente e automatizar com agentes de IA.

## Objetivos

- Implementar governança de software: code review, ADRs, compliance
- Criar agentes de IA especializados e reutilizáveis
- Auditar sistemas contra 16 dimensões de qualidade
- Automatizar todo o ciclo de desenvolvimento com pipelines e agentes

## Público-alvo

Tech leads, arquitetos e desenvolvedores seniores responsáveis por qualidade e evolução do sistema.
