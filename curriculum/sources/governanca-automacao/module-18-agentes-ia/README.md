## Biblioteca de Agentes

A formação inclui 17 agentes pré-construídos na [Biblioteca de Agentes](../../biblioteca-agentes/README.md):

### Produto
- `product-discovery-agent` — Discovery, User Stories, RICE
- `ux-research-agent` — Pesquisa, entrevistas, personas

### Design
- `ux-designer-agent` — User flow, wireframes, acessibilidade
- `ui-designer-agent` — Design System, tokens, dark mode

### Arquitetura
- `enterprise-architect-agent` — Clean Arch, DDD, ADRs
- `database-architect-agent` — Modelagem, índices, partições

### Desenvolvimento
- `backend-expert-agent` — NestJS, APIs, DDD
- `frontend-expert-agent` — Next.js, RSC, Tailwind
- `prisma-expert-agent` — Schema, migrations, queries

### Infraestrutura
- `devops-expert-agent` — Docker, CI/CD, deploy
- `security-expert-agent` — OWASP, JWT, rate limiting

### Qualidade
- `qa-expert-agent` — Testes, Playwright, cobertura
- `performance-expert-agent` — Otimização, caching, Web Vitals

### Governança
- `auditor-agent` — 16 tipos de auditoria com score
- `documentation-agent` — ADRs, README, Swagger
- `refactoring-agent` — Refatoração guiada por regras
