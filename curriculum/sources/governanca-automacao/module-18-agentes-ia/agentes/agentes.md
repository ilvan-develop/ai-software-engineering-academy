# Agentes para o Módulo 18

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar o conteúdo sobre criação de agentes |
| Technical Writer | Escrever a aula sobre agentes especializados |
| Reviewer | Validar precisão técnica dos conceitos |
| Slide Creator | Criar apresentação visual sobre agentes |

## Instruções específicas

### Curriculum Architect

Ao planejar este módulo, foque em:
- Demonstrar a diferença prática entre agente genérico e especializado
- Ensinar a anatomia replicável de um agente
- Mostrar como combinar agentes em pipelines
- Conectar com a biblioteca de agentes da formação

### Technical Writer

Ao escrever este módulo:
- Tom prático e acionável ("como fazer", não apenas "o que é")
- Incluir exemplos de prompts bons vs. ruins
- Mostrar código real de configuração (opencode.json)
- Usar diagramas ASCII para pipelines

### Reviewer

Ao revisar este módulo:
- Os exemplos de prompts seguem as boas práticas ensinadas?
- A estrutura do agente está consistente com a biblioteca real?
- Os pipelines fazem sentido na ordem apresentada?
- Há exemplos práticos suficientes?
