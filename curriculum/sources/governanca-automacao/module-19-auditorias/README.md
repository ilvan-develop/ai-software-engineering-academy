## Tipos de Auditoria

- Arquitetura, Segurança, Frontend, Backend, UX, UI
- Banco de Dados, APIs, Performance, DevOps
- Governança, Multi-Tenant, Código, Dependências, TypeScript

## Saída de cada auditoria

- Relatório completo com score (0-10)
- Riscos classificados (Blocker, Critical, Major, Minor)
- Plano de ação priorizado
- Recomendações estratégicas
