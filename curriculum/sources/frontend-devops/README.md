# Frontend & DevOps

**Levar o sistema ao usuário e mantê-lo funcionando.** Frontend moderno, infraestrutura, qualidade e observabilidade.

## Objetivos

- Desenvolver frontends performáticos com Next.js e React
- Containerizar aplicações com Docker e orquestrar com Kubernetes
- Implementar pipelines CI/CD e estratégias de deploy
- Garantir qualidade com testes unitários, integração e E2E
- Monitorar sistemas com logs, métricas e tracing (OpenTelemetry, Grafana)

## Público-alvo

Desenvolvedores fullstack que querem dominar tanto o frontend quanto a operação do sistema em produção.
