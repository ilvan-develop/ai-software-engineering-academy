# Agentes para o Módulo 11

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar conteúdo frontend |
| Technical Writer | Exemplos reais de Next.js |
| Reviewer | Validar boas práticas |
| Frontend Expert Agent | Referência prática |

## Instruções específicas

### Technical Writer

Ao escrever:
- Código funcional de Next.js App Router (não pages router)
- Server Components como padrão, Client Components só quando necessário
- Incluir acessibilidade (ARIA, teclado) em todos os exemplos
- Mostrar responsividade com Tailwind
