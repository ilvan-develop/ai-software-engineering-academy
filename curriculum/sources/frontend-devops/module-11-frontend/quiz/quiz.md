# Quiz — Módulo 11

## Pergunta 1

Qual a regra de ouro para escolher entre Server Component e Client Component?

- a) Sempre usar Client Component
- b) Server Component por padrão, Client apenas quando precisar de interatividade
- c) Sempre usar Server Component
- d) Depende do humor do desenvolvedor

**Resposta:** b

---

## Pergunta 2

O que um Server Component NÃO pode fazer?

- a) Acessar o banco de dados
- b) Usar useState ou useEffect
- c) Renderizar HTML
- d) Ler variáveis de ambiente

**Resposta:** b
