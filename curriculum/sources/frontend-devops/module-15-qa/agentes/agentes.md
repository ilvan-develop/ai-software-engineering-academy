# Agentes para o Módulo 15

## Agentes envolvidos

| Agente | Função no módulo |
|--------|------------------|
| Curriculum Architect | Estruturar conteúdo de QA |
| Technical Writer | Escrever exemplos de testes reais |
| Reviewer | Validar boas práticas de teste |
| QA Expert Agent | Referência prática |

## Instruções específicas

### Technical Writer

Ao escrever:
- Exemplos completos de Jest (describe, it, expect, mocks)
- Testes de integração com Supertest (app.getHttpServer)
- Testes E2E com Playwright (page.goto, fill, click)
- Mostrar TDD na prática (Red → Green → Refactor)
