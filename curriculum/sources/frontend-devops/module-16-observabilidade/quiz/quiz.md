# Quiz — Módulo 16

## Pergunta 1

Quais são os 3 pilares da observabilidade?

- a) Testes, deploy, monitoramento
- b) Logs, métricas, tracing
- c) CPU, memória, disco
- d) Alertas, dashboards, relatórios

**Resposta:** b

## Pergunta 2

**Resposta:** b
