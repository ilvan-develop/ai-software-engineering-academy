# Agentes para o Módulo 04

## Agentes envolvidos

| Agente | Função |
|--------|--------|
| Curriculum Architect | Estruturar UX do básico ao avançado |
| UX Research Agent | Conteúdo de pesquisa qualitativa e quantitativa |
| Accessibility Specialist | Diretrizes WCAG e boas práticas de a11y |
| Technical Writer | Exemplos de código acessível e UX Writing |
| Frontend Developer Agent | Validação prática dos exemplos React/TypeScript |

## Instruções específicas

### UX Research Agent

Ao escrever:
- Incluir exemplos de roteiros de entrevista, teste de usabilidade e card sorting
- Mostrar os 5 planos de Garrett com exemplos concretos para devs
- Diferenciar proto-personas de data-driven personas
- Incluir o conceito de anti-personas

### Accessibility Specialist

Ao escrever:
- Reforçar que acessibilidade não é feature, é requisito
- Incluir exemplos de HTML semântico vs não semântico
- Mostrar níveis WCAG (A, AA, AAA) com exemplos práticos
- Incluir ARIA labels, navegação por teclado e contraste

### Technical Writer

Ao escrever:
- Exemplos de código em TypeScript/React para componentes acessíveis
- Centralização de mensagens de erro com UX Writing
- Feature flags baseadas em personas
- Checklist de acessibilidade implementável em código

### Frontend Developer Agent

Ao escrever:
- Validar que exemplos de código seguem boas práticas de acessibilidade
- Garantir que exemplos de HTML semântico funcionam com screen readers
- Verificar que componentes contemplam estados: loading, vazio, erro, sucesso
