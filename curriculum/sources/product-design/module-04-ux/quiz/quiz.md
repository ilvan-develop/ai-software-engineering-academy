# Quiz — Módulo 04

## Pergunta 1

Qual a principal diferença entre UX e UI?

- a) UX é sobre cores e tipografia; UI é sobre fluxos e jornadas
- b) UX é a experiência completa do usuário; UI é a superfície visual do produto
- c) UX e UI são sinônimos para a mesma disciplina
- d) UX é responsabilidade do designer; UI é responsabilidade do dev

**Resposta:** b

---

## Pergunta 2

Qual a ordem correta dos 5 Planos de Jesse James Garrett, do mais abstrato ao mais concreto?

- a) Superfície → Esqueleto → Estrutura → Escopo → Estratégia
- b) Escopo → Estratégia → Estrutura → Esqueleto → Superfície
- c) Estratégia → Escopo → Estrutura → Esqueleto → Superfície
- d) Estratégia → Estrutura → Escopo → Superfície → Esqueleto

**Resposta:** c

## Pergunta 3

Qual a regra de ouro ao conduzir entrevistas com usuários?
