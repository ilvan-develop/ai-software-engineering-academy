# Agentes para o Módulo 05

## Agentes envolvidos

| Agente | Função |
|--------|--------|
| Curriculum Architect | Estruturar conteúdo de wireframing |
| UX Designer Agent | Referência em técnicas e ferramentas de wireframe |
| Technical Writer | Exemplos de código e especificações de handoff |

## Instruções específicas

### UX Designer Agent

- Fornecer exemplos reais de wireframes em diferentes níveis de fidelidade (baixa, média, alta)
- Explicar as diferenças entre wireframe, mockup e protótipo com casos práticos
- Demonstrar técnicas como Crazy 8s, Grey Box e Wireflow com exemplos visuais
- Sugerir ferramentas gratuitas e pagas para wireframing, com prós e contras de cada uma
- Incluir boas práticas de documentação de wireframes para contextos enterprise

### Technical Writer

- Exemplos de código TypeScript que representem estruturas de wireframe (interfaces, tipos, especificações)
- Demonstrar como documentar wireframes programaticamente (metadados, versionamento, aprovações)
- Incluir especificações de handoff que conectam wireframe ao código (grid, breakpoints, estados, contratos de API)
- Ilustrar a anatomia de uma tela com tipos bem definidos para cada elemento (header, hero, CTA, footer)
