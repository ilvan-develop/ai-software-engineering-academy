# Agentes para o Módulo 03

## Agentes envolvidos

| Agente | Função |
|--------|--------|
| Curriculum Architect | Estruturar Design Thinking no currículo |
| UX Researcher Agent | Referência para técnicas de empatia e entrevistas |
| Product Designer Agent | Exemplos de prototipação e ferramentas de design |
| Agile Coach Agent | Integração com Scrum/Kanban |

## Instruções específicas

### UX Researcher Agent

Ao escrever:
- Incluir roteiros reais de entrevista com perguntas abertas
- Mostrar exemplos de mapas de empatia e como preenchê-los
- Detalhar técnicas de observação contextual
- Explicar a diferença entre o que o usuário diz e o que faz

### Product Designer Agent

Ao escrever:
- Mostrar exemplos visuais de protótipos de baixa e alta fidelidade
- Incluir matriz de priorização impacto x esforço
- Demonstrar Crazy 8 com exemplos práticos
- Sugerir ferramentas (Figma, Miro, Balsamiq, papel)

### Agile Coach Agent

Ao escrever:
- Explicar o modelo two-track (Discovery + Delivery)
- Mostrar como adaptar Kanban com coluna de Discovery
- Fornecer template de Discovery Sprint
- Incluir métricas de aprendizado vs. entrega
