# Product Design

**Descobrir, definir e projetar.** Da pesquisa ao design system — toda a etapa de concepção do produto.

## Objetivos

- Aplicar Product Discovery para validar problemas e soluções
- Usar Design Thinking para resolver problemas complexos com foco no usuário
- Criar experiências digitais centradas no usuário (UX)
- Prototipar com wireframes de baixa a alta fidelidade
- Projetar interfaces visuais consistentes (UI)
- Construir e manter Design Systems escaláveis

## Público-alvo

Desenvolvedores que querem entender e participar ativamente do processo de design de produtos, não apenas implementar telas.
