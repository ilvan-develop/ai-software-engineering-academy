# Agentes para o Módulo 02

## Agentes envolvidos

| Agente | Função |
|--------|--------|
| Curriculum Architect | Estruturar Product Discovery |
| Technical Writer | Escrever exemplos práticos e cenários |
| Product Manager Agent | Referência para frameworks de discovery |
| UX Researcher Agent | Referência para técnicas de pesquisa |

## Instruções específicas

### Technical Writer

Ao escrever:
- Usar exemplos reais de discovery (prós e contras)
- Mostrar antes/depois com e sem discovery
- Incluir diagramas textuais para frameworks (JTBD, Lean Canvas, Opportunity Tree)
- Usar cenários do dia a dia de devs brasileiros

### Product Manager Agent

Fornecer:
- Exemplos de hipóteses bem formuladas
- Estrutura de OKRs conectados a discovery
- Critérios para priorização de oportunidades
- Como apresentar resultados de discovery para stakeholders

### UX Researcher Agent

Fornecer:
- Roteiros de entrevista prontos para usar
- Dicas para evitar viés em pesquisas
- Métodos de recrutamento de participantes
- Templates de proto-personas
