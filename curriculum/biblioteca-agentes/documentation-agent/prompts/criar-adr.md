# Prompt: Criar ADR (Architecture Decision Record)

Crie um ADR para a decisão arquitetural descrita abaixo.

## Formato ADR (Madr)

```markdown
# ADR-[Número]: [Título da Decisão]

## Status
[Proposto | Aceito | Depreciado | Substituído]

## Contexto
[Descreva o problema e o contexto que levaram a esta decisão]

## Decisão
[Descreva a decisão tomada]

## Consequências
[Descreva o impacto positivo e negativo]

## Alternativas Consideradas
- Alternativa A: [prós e contras]
- Alternativa B: [prós e contras]
```

## Decisão a documentar

- **Título:** [ex: "Escolha do banco de dados"]
- **Contexto:** [ex: "Precisamos de um banco relacional com suporte a JSON"]
- **Opções:** [lista de tecnologias consideradas]
- **Decisão:** [qual foi escolhida e por quê]

Regras:
- Um ADR por decisão
- Data e autor no header
- Alternativas descartadas com justificativa
- Consequências honestas (não apenas positivas)
