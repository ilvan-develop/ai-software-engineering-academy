# Workflow: Documentation Agent

```
1. Analisar código e arquitetura existente
2. Identificar documentação necessária (lacunas)
3. Criar/atualizar README.md
4. Criar ARCHITECTURE.md com diagramas C4
5. Documentar APIs (Swagger/OpenAPI)
6. Escrever/atualizar ADRs
7. Criar CONTRIBUTING.md
8. Revisar consistência e links
```
