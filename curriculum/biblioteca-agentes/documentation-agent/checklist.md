# Checklist: Documentation Agent

## README
- [ ] Título e descrição do projeto
- [ ] Pré-requisitos (Node, Docker, etc.)
- [ ] Passos de setup (clonar, instalar, configurar, rodar)
- [ ] Comandos disponíveis (dev, build, test, lint)
- [ ] Estrutura de diretórios explicada
- [ ] Variáveis de ambiente documentadas (.env.example)
- [ ] Link para documentação adicional

## Arquitetura
- [ ] Diagrama C4 (Contexto + Container)
- [ ] Decisões arquiteturais documentadas (ADRs)
- [ ] Stack tecnológica justificada
- [ ] Fluxo de dados principais descrito

## API
- [ ] Swagger/OpenAPI configurado
- [ ] Todos os endpoints documentados
- [ ] Exemplos de request/response
- [ ] Autenticação documentada
- [ ] Códigos de erro documentados

## Contribuição
- [ ] CONTRIBUTING.md com guia de PRs
- [ ] Padrões de commit (conventional commits)
- [ ] Padrões de código (linter, formatação)
