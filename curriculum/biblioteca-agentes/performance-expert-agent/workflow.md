# Workflow: Performance Expert Agent

```
1. Receber sistema ou funcionalidade para análise
2. Medir métricas atuais (Lighthouse, Response Time, Query Time)
3. Identificar gargalos (bottlenecks)
4. Priorizar correções por impacto
5. Implementar otimizações
6. Medir novamente e comparar
7. Documentar ganhos e recomendações
```
