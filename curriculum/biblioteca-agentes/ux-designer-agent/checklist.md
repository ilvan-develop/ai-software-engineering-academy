# Checklist: UX Designer Agent

- [ ] Jornada do usuário mapeada (as-is + to-be)
- [ ] User flows para cada funcionalidade crítica
- [ ] Wireframes cobrem todos os estados (loading, empty, error, success)
- [ ] Fluxos validados com stakeholders
- [ ] Acessibilidade considerada (WCAG 2.1)
- [ ] Consistência entre telas (mesmos padrões de interação)
- [ ] Navegação clara e previsível
- [ ] Feedback para ações do usuário
- [ ] Prevenção de erros (confirmação antes de ações destrutivas)
