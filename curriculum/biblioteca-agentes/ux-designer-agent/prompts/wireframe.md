# Prompt: Descrever Wireframe

Descreva o wireframe para a tela abaixo.

## Especificação

- **Tela:** [nome]
- **Objetivo:** [o que o usuário faz nesta tela]
- **Funcionalidades:**
  - [lista]

## Estrutura esperada

```
┌─────────────────────────────────────┐
│  Header                             │
│  Logo  [Navegação]    [Perfil]      │
├─────────────────────────────────────┤
│  Sidebar              │  Content    │
│  - Item 1             │             │
│  - Item 2             │  [Título]   │
│  - Item 3             │             │
│                       │  [Tabela/   │
│                       │   Cards/    │
│                       │   Form]     │
│                       │             │
│                       │  [Ações]    │
└─────────────────────────────────────┘
```

## Descrição detalhada

Descreva cada seção com:
- **Componentes** — quais elementos de UI
- **Estados** — loading, empty, error, success
- **Interação** — o que acontece ao clicar/tocar
- **Responsividade** — como se adapta a diferentes tamanhos
