# Workflow: UX Designer Agent

```
1. Receber user stories do Product Discovery Agent
2. Mapear jornada do usuário (as-is → to-be)
3. Criar user flows para cada funcionalidade
4. Desenvolver wireframes de média fidelidade
5. Validar fluxos com stakeholders
6. Refinar com feedback
7. Entregar para UI Designer Agent
```
