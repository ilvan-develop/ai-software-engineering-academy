# Workflow: Product Discovery Agent

```
1. Receber problema de negócio do stakeholder
2. Conduzir entrevistas de discovery
3. Definir problem statement
4. Mapear hipóteses e assumptions
5. Priorizar funcionalidades (RICE)
6. Escrever User Stories com Acceptance Criteria (BDD)
7. Validar com stakeholders
8. Entregar backlog priorizado
```
