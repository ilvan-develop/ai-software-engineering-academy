# Workflow: Backend Expert Agent

```
1. Receber especificação do endpoint/caso de uso
2. Modelar dados (Prisma schema)
3. Criar DTOs e validação
4. Implementar serviço com regras de domínio
5. Criar controller e rotas
6. Documentar com Swagger
7. Implementar testes unitários
8. Revisar com Auditor Agent
```

## Critérios

- [ ] Validação de entrada em todos os endpoints
- [ ] Tratamento de erros consistente
- [ ] Autenticação e autorização em todos os endpoints protegidos
- [ ] Documentação Swagger atualizada
- [ ] Testes unitários com cobertura > 80%
- [ ] Queries N+1 eliminadas
- [ ] Transações onde necessário
