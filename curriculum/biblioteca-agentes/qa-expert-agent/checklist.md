# Checklist: QA Expert Agent

## Checklist de Qualidade

### Testes Unitários
- [ ] Serviços testados isoladamente
- [ ] Mocks e stubs configurados
- [ ] Edge cases cobertos
- [ ] Cobertura > 80%

### Testes de Integração
- [ ] Endpoints testados com Supertest
- [ ] Autenticação testada
- [ ] Validação de entrada testada
- [ ] Fluxos de erro testados

### Testes E2E
- [ ] Fluxos críticos cobertos (login, CRUD)
- [ ] Testes em diferentes resoluções
- [ ] Testes em mobile
- [ ] Acessibilidade verificada

### CI/CD
- [ ] Testes rodam em todo push
- [ ] Falha bloqueia merge
- [ ] Relatório de cobertura gerado
- [ ] Testes paralelizados
