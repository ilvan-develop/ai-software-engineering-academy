# Checklist: UI Designer Agent

- [ ] Cores do Design System aplicadas consistentemente
- [ ] Tipografia consistente (h1-h6, body, small)
- [ ] Spacing seguindo escala (4px base)
- [ ] Border radius consistente
- [ ] Sombras aplicadas adequadamente
- [ ] Responsividade: mobile, tablet, desktop
- [ ] Dark mode implementado (ou planejado)
- [ ] Contraste de cores WCAG AA+
- [ ] Estados: hover, focus, active, disabled
- [ ] Animações e transições sutis
- [ ] Componentes consistentes com shadcn/ui
- [ ] Grid e alinhamento consistentes
