# AI Enterprise Software Engineering

**Da ideia ao sistema empresarial utilizando agentes de Inteligência Artificial**

Uma formação completa que ensina **como pensar como uma equipe Enterprise** — combinando princípios consagrados de engenharia de software com o poder dos agentes de IA.

## Cursos

### 1. [Fundamentos Enterprise](cursos/fundamentos-enterprise/README.md)
A base de tudo — mentalidade, papéis e divisão de trabalho entre humanos e IA.

| # | Módulo | Foco |
|---|--------|------|
| 00 | **Introdução** | Papel do arquiteto, PO, IA — divisão de trabalho |
| 01 | **Mentalidade Enterprise** | Escalabilidade, governança, segurança, compliance |

### 2. [Product Design](cursos/product-design/README.md)
Da descoberta ao design system — toda a concepção do produto.

| # | Módulo | Foco |
|---|--------|------|
| 02 | **Product Discovery** | Problem → Solution → MVP → Roadmap |
| 03 | **Design Thinking** | Empatia, pesquisa, personas, hipóteses |
| 04 | **UX** | Research, user flow, arquitetura da informação |
| 05 | **Wireframes** | Baixa → Média → Alta fidelidade |
| 06 | **UI Design** | Tipografia, grid, cores, tokens |
| 07 | **Design System** | Componentes, variantes, dark mode, acessibilidade |

### 3. [Arquitetura Backend](cursos/arquitetura-backend/README.md)
Arquitetura, modelagem, APIs, segurança e multi-tenancy.

| # | Módulo | Foco |
|---|--------|------|
| 08 | **Arquitetura** | Clean Arch, DDD, SOLID, Hexagonal, Event Driven |
| 09 | **Modelagem** | ERD, UML, diagramas, fluxos |
| 10 | **Backend** | NestJS, Prisma, PostgreSQL, APIs |
| 12 | **Segurança** | OWASP, JWT, OAuth, Rate Limit |
| 13 | **Multi-Tenant** | Isolamento, billing, feature flags |

### 4. [Frontend & DevOps](cursos/frontend-devops/README.md)
Frontend moderno, infraestrutura, qualidade e observabilidade.

| # | Módulo | Foco |
|---|--------|------|
| 11 | **Frontend** | Next.js, React, Server Components |
| 14 | **DevOps** | Docker, CI/CD, Kubernetes |
| 15 | **QA** | Testes unitários, integração, E2E |
| 16 | **Observabilidade** | OpenTelemetry, Grafana, Prometheus |

### 5. [Governança & Automação](cursos/governanca-automacao/README.md)
Governança, agentes de IA, auditorias e automação do ciclo de desenvolvimento.

| # | Módulo | Foco |
|---|--------|------|
| 17 | **Governança** | Git Flow, ADR, RFC, Code Review |
| 18 | **Agentes de IA** | Criação de agentes especializados |
| 19 | **Auditorias** | 16 tipos de auditoria com score e plano de ação |
| 20 | **Automação** | Workflow integrado de agentes |

### 6. [Capstone — Projeto Final](cursos/capstone/README.md)
Construir um SaaS Enterprise completo, do discovery ao deploy.

| # | Módulo | Foco |
|---|--------|------|
| 21 | **Projeto Final** | Plataforma SaaS com multi-tenancy, pagamentos, observabilidade |
